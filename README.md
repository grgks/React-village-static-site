# Ποταμούλα | Potamoula

Static tourist website for the village of Potamoula, Aetolia-Acarnania, Greece.

🌐 **Live site**: [potamoula.gr](https://www.potamoula.gr)

## Features
- Bilingual (Greek / English) with react-i18next
- Image carousel with auto-slide and lightbox
- Google Maps integration
- Responsive design (mobile friendly)
- SEO optimized (meta tags, sitemap, Google Search Console)

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- react-router-dom
- react-i18next

## Deployment
- Hosted on **Vercel**
- Domain: **potamoula.gr** (Papaki)
